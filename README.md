# GitHub Finder

A simple web app to search GitHub profiles and repositories using the GitHub API.

## 🚀 Features
- Search GitHub users by username
- View profile details like avatar, bio, followers, following
- Display list of repositories

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- GitHub API
